# **App Name**: QuickPay QR Generator

## Core Features:

- Amount Entry: User inputs the amount for which the QR code needs to be generated. Validation is included to check amount is a positive value.
- QR Code Generation: Generates a UPI QR code based on the entered amount and the pre-defined UPI ID (ap.raamachandiran@oksbi) with a payee name of Sendhur Juice and Snacks and an intent message.
- QR Code Display: Displays the generated QR code on the screen, with the linked UPI payment details for user confirmation.
- Download QR Code: Allows the user to download the generated QR code as an image file for sharing or printing.

## Style Guidelines:

- Primary color: A vibrant orange (#FF7900) to convey energy and action, in relation to sales.
- Background color: Light orange (#FFD3A7), providing a bright and clean backdrop with low saturation (25%).
- Accent color: Yellow-orange (#FF9F00), analogous to the primary, adding emphasis to interactive elements.
- Font: 'Inter' (sans-serif) for both headings and body text. Note: currently only Google Fonts are supported.
- Use clean, minimalist icons for actions like 'download' and 'refresh QR code.'
- Simple, single-column layout optimized for mobile viewing. Prominent display of the generated QR code.
- Subtle fade-in animation when the QR code is generated or updated.
